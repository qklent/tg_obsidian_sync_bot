# Agent Delegation Rules

- Read `notes/_codemap.md` first for codebase orientation before deep exploration
- Use Opus for tasks requiring deep reasoning (planning, clarification, coding)
- Use Sonnet for mechanical tasks (testing, reviewing, git operations, docs)
- Minimize tool access per agent — only what it needs
- Run `/compact` between planning and coding phases to free context
- Never let agents modify files outside their designated scope
